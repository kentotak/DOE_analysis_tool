# DOE_analysis_tool
Matlab application for the analysis of DOE results.

## Download
Keep the same file structure when you download the files.

## Usage
Run DOE_analysis_tool.mlapp in Matlab. You can change the depth interval. Press the "Plot RSquared" button to update the R squared plot. You can also change the model, either by using one of the predefined Matlab models, or by using the "Personalized" option that allows you to directly change the residual stress equation.
